# Webgoat 8

## Docker
Download webgoat image
```docker pull webgoat/webgoat-8.0```

Run webgoat docker
```docker run -p 8080:8080 -t webgoat/webgoat-8.0```

Accessing webgoat page
```http://127.0.0.1:8080/WebGoat```

Find the webgoat container id
```docker ps```

Stop webgoat docker
```docker stop 1240f272bc03```
